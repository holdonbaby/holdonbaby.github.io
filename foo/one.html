# 什么是Redis?

Redis是一个开源的、可联网的、在内存中的、kv可选持续时间的数据库。

他用标准C写的。

今天Redis的发展协议由Redis实验室支持。

再此之前，他由Pivtoal和VMware支持。

根据数据库引擎网站每月的评分统计，Redis是世界上最受欢迎的kv数据库。

Redis的全称是 远程字典服务。

# 什么是Apache Kafka?

Apache Kafka是一个分布式流平台，用作构建实时管道，可以作为一个消息broker(这里应该翻译成 代理人或者是中间人)或者作为大数据应用的日志聚合解决方案的代替。

# 什么是Tomcat?

Apache Tomcat 是一个开源的 Web服务和 小服务开发容器，他是 Apache基础软件项目。Tomcat应用了Java轻量服务和Java服务由Oracle开发，并提供用java代码开发了一个纯粹java的http web服务环境。简单的配置tomcat运行在一个单一的操作系统进程中。这个进程运行 一个java虚拟机。每一个来自浏览器的单独的http请求都在tomcat进程中单独线程处理。

# 什么是Apache Zookeeper?

Apache Zookeeper是Apache软件基础项目，提供了一个开源的、分布式的配置（）服务，同步服务。并且注册给了大量分布式系统中。ZooKeeper是Hadoop的子项目，但现在他本身就是更高级的项目。

# 什么是ubuntu?

ubuntu是一种基于Debian的操作系统，他可以运行在云端并与网络连接。

他是世界上最流行的公有云端和开源栈的操作系统。

他是第一容器平台。

从docker到k8s再到LXD，ubuntu可以规模化运行你的容器。

ubuntu支持世界上成千上万的电脑终端，他是快速，安全简单的。

由规范的有限公司主导ubuntu的发展。

规范公司会提供大规模的技术支持和其他有关ubuntu的服务。
