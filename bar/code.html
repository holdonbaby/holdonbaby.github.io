# 致心里没底的大伙

后端写代码，是为了处理信息。

func add(a,b int)int{
    return a+b
}

一个简单加法，就是对信息的处理。更加复杂的后端，就是许多个简单的加法函数嵌套。

作为非科班的我们，需要做的是，接受计算机中的一些概念，最好从中文/英语字眼出发去理解含义。

  • server和service
  • dataBased和dataStore
  • handle\句柄
